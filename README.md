# BomjWiFi_Repeater  (NAT Router)
 Простой WiFi репитер на базе ESP8266 с возможностью настройки через веб-интерфейс. Устройство подключается к существующей WiFi сети и создает свою точку доступа, расширяя зону покрытия.

## Что умеет:
- Подключается к существующему WiFi роутеру
- Создает свою точку доступа с любым именем и паролем
- Передает интернет клиентам (телефон, ноутбук, ТВ-приставка)
- Имеет веб-интерфейс для настройки
- Сохраняет настройки в памяти
- Автоматически переподключается при обрыве связи

## Реальные показатели:
- Скорость: до 2-5 Мбит/с
- Радиус: до 30-50 метров (в зависимости от условий)
- Клиентов: до 4 устройств одновременно (Можно увеличить до 8)

## Как это работает:
- Загружаем прошивку в ESP8266
- Подключаемся к сети ESP8266_Repeater_Setup
- Заходим на 192.168.4.1
- Вводим данные своего WiFi и настройки новой сети
- Готово! Репитер работает.

## Важно!!! В файле platformio.ini пропишите две эти строчки:
```ini
[env:nodemcuv2]
platform = espressif8266
....
board_build.f_cpu = 160000000L
build_flags = -D PIO_FRAMEWORK_ARDUINO_LWIP2_HIGHER_BANDWIDTH
```
